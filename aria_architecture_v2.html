intake pipeline
Intake agents (01–03) — trigger on question submission
Agent 01
Gatekeeper
Screens every question the moment it is submitted. Filters harmful, obnoxious, and bad-faith content. Returns approve / revise / reject + reason code. First gate — nothing passes without clearance.
On: question submitted
Agent 02
Question Rater
Scores on clarity, depth, originality, and engagement potential. Assigns Quality Score 0–100. Low scores return with specific improvement prompts before publishing to the feed.
On: question approved
Agent 03
Smart Matcher
Reads the question, scans declared expertise, answer history, and subject tags. Sends personalised invitations to the top 5 best-fit answerers only. Never cold-broadcasts to the full community.
On: question published
answer arrives → 6 atomic evaluators fire in parallel
Atomic evaluation agents (04–09) — one index each, all parallel, all independent
Agent 04
Engagement Indexer
Engagement Index (EI) · weight 25%
Does the answer make the reader lean in? Scores narrative pull, hook strength, questions raised, energy of language, and degree to which it invites further thinking. Answers that engage deeply score above 80.
On: answer submitted
Agent 05
Retention Indexer
Retention Index (RI) · weight 20%
Will this be remembered in 7 days? Scores use of memorable anchors, structural repetition, analogy density, and quotability of key phrases. Forgettable, generic answers score below 50.
On: answer submitted
Agent 06
Applicability Indexer
Real-life Applicability Index (RAI) · weight 25%
Can this be used on Monday morning? Scores context specificity, transferability, step clarity, and whether the answer bridges theory to practice. Purely abstract answers are capped at 40.
On: answer submitted
Agent 07
Creativity Indexer
Creativity & Originality Index (COI) · weight 15%
Is this a fresh perspective or a restatement? Scores novelty of framing, unexpected connections, metaphor originality, and divergence from the most predictable response. Safe answers cap at 60.
On: answer submitted
Agent 08
Accuracy Verifier
Accuracy Index (AI) · weight 15%
Is the answer factually correct and well-grounded? Scores claim verifiability, logical coherence, and appropriate grounding in evidence or experience. Penalises demonstrable factual errors.
On: answer submitted
Agent 09
Integrity Guard
Integrity Flag (IF) · binary gate
Runs fully independently. Checks for plagiarism, hallucination, citation abuse, and bad faith. A flag suspends the answer. An IF overrides all index scores regardless of their values. Cannot be appealed without independent evidence.
On: answer submitted (parallel)
indices → ARIA aggregates → Arbiter reconciles dual grades
Grade arbiter (Agent 10) — produces the Composite Defended Score (CDS)
Agent 10
Comparative Grade Arbiter
Receives all 5 index scores from Agents 04–08. Computes the Algorithmic Grade (AG). Receives the Engageneer™ Peer Grade (PG) with mandatory written justification per criterion. Blends both into the Composite Defended Score. Raises a Divergence Flag if AG and PG differ by more than 15 points — triggering mandatory second Engageneer™ review. Every CDS is fully explainable, challengeable, and permanently auditable.
On: all indices received + Engageneer™ PG submitted
The Composite Defended Score (CDS) — published, transparent formula
AG = (EI × 0.25) + (RI × 0.20) + (RAI × 0.25) + (COI × 0.15) + (AI × 0.15)
CDS = (AG × 0.60) + (PG × 0.40)
IF integrity flag raised → CDS suspended pending ARIA review
IF |AG − PG| > 15 → Divergence Flag → second Engageneer™ review mandatory
Every weight is published and stable. Every index is explained in plain language. Every CDS can be broken into its five components on request by the answerer. This formula is identical for every answer, every answerer, and every tier level. No exceptions. No special cases.
Algorithmic Grade (AG)
Engageneering™ AI Grade
Produced by Agents 04–08. Objective, instant, and consistent across all answers. Broken into 5 published indices. Cannot be influenced by personal relationships, reputation, or bias. Can be challenged only with evidence of a specific indexing error.
Peer Grade (PG)
Engageneer™ Human Grade
Produced by a certified Engageneer™. Mandatory written justification required per criterion. Adds contextual wisdom, nuance, and lived classroom experience that no algorithm can fully replicate. Subject to divergence review and public visibility on request.
CDS finalised → growth agents → tier escalation
Growth agents (11–12) — trigger on CDS finalisation & weekly cadence
Agent 11
Tier Engine
Monitors cumulative CDS history, question quality scores, mentorship activity, and time-in-tier per user. On threshold crossing, calculates eligibility, presents evidence to ARIA, and triggers promotion or demotion — with full audit trail available to the user on request at any time.
On: CDS finalised · milestone thresholds
Agent 12
Insight Narrator
Generates personalised weekly reports per user — full index breakdowns, growth edges, comparison to the user's own past scores (never to others), and specific next-action recommendations written in the Engageneering™ voice. Every word earned from real activity data.
On: weekly cadence per user
