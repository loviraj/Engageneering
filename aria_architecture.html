Intake agents — trigger on submission
Agent 01
Gatekeeper
Screens every question the moment it is submitted. Filters harmful, obnoxious, and bad-faith content. Returns approve / revise / reject + reason code. Zero human review needed.
On: question submitted
Agent 02
Question Rater
Scores approved questions on clarity, depth, originality, and engagement potential. Assigns Quality Score 0–100. Low scores return with specific improvement prompts before publishing.
On: question approved
Agent 03
Smart Matcher
Reads the question. Scans community expertise, past answer ratings, and subject tags. Sends personalised invitations to the top 5 best-fit answerers only. Never cold-blasts everyone.
On: question published
answers arrive
Answer agents — run in parallel on every response
Agent 04
Answer Evaluator
Scores on six axes: accuracy, engagement quality, analogy use, depth, originality, and practical applicability. Produces composite Engagement Score 0–100. Determines ranking and reward eligibility.
On: answer submitted
Agent 05
Integrity Guard
Independently checks for plagiarism, hallucination, citation accuracy, and bad-faith content. Runs parallel to Evaluator — never combined. Flags silently. ARIA adjudicates conflicts between agent outputs.
On: answer submitted
scores to ARIA
Growth agents — trigger on thresholds & cadence
Agent 06
Tier Engine
Monitors every user's cumulative signal — question quality, answer scores, mentorship, time-in-tier. On threshold crossing, calculates eligibility, presents evidence to ARIA, triggers promotion or demotion with full audit trail.
On: score thresholds & milestones
Agent 07
Insight Narrator
Generates personalised weekly growth reports per user — what landed, what improved, which answers changed someone's thinking, what the next growth edge is. Written in the Engageneering™ voice. Every insight earned from real data.
On: weekly cadence per user
